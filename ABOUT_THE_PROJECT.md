# About This Project

This repository contains the final project developed for the Data Science in the Life Sciences program at Freie Universität Berlin.

## Project Goal

To investigate whether integrated RNA-seq and DNA methylation data can be used to classify molecular subtypes of Lung Adenocarcinoma (LUAD).
