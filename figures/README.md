# Figures

This directory contains figures generated throughout the analysis workflow.

The figures document key stages of the project, including:

* Data exploration and quality assessment
* Multi-omics integration
* Machine learning model performance
* Feature importance analyses
* Functional enrichment results
* Biological interpretation of predictive biomarkers

Selected figures are displayed in the main repository README to provide a visual overview of the workflow and results.
